MORI는 명함 스캔으로 바이어를 모으고, 관심도·신뢰도를 분석해, 개인화된 메시지와 브로셔로 첫 거래를 만드는 AI 기반 글로벌 B2B 세일즈 인텔리전스 플랫폼입니다. 이 가이드는 프로젝트 생성부터 바이어 관리, 메시지 발송까지 실제 운영 흐름을 그대로 따라갑니다.

MORI 팀 — 5분 만에 듣는 MORI 시작 가이드
▶ MORI AUDIO

한눈에 보는 MORI

  1. 명함 OCR은 무료입니다. 카메라 촬영 또는 파일 업로드로 이름·회사·이메일을 자동 수집하고, 선택한 박람회 리스트에 저장합니다.
  2. Interest Engine × Trust Engine. 관심도·신뢰도를 각각 0~100으로 산출해 바이어를 즉시 연락 / 검토 필요 / 장기 육성으로 자동 분류합니다.
  3. 개인화 발송은 1크레딧. 이름·회사 등 7종 변수를 자동 치환하며, 1:1 메시지 1건당 1크레딧이 차감됩니다.
  4. 브로셔는 AI가 추천합니다. 분류·언어별로 자료를 관리하고, 바이어에게 맞는 자료를 자동 제안하거나 메시지에 첨부합니다.
  5. 시작은 무료입니다. 이메일 연동 시 웰컴 50크레딧이 지급되고, 명함 OCR·KPI 대시보드·프로젝트 생성은 무료로 제공됩니다.
01 — Getting Started

MORI 시작하기

MORI를 시작할 준비가 되었습니다. 이메일 연결, 바이어 업로드, 브로셔 등록 세 가지를 완료하면 홈에서 후속 필요 리드, 신호 상승, 브로셔 반응, AI 추천 액션을 한눈에 확인할 수 있습니다.

로그인은 비밀번호 없이 이메일 OTP(6자리, 10분 유효) 방식이며, Gmail·Outlook 계정을 OAuth로 연동해 실제 발송 계정을 설정합니다. 프로필을 완료하면 운영 홈으로 진입합니다.

시작 가이드 · 3단계+

① 이메일 연결 — Gmail / Outlook 계정 연동  ·  ② 바이어 업로드 — CSV 또는 명함 촬영  ·  ③ 브로셔 등록 — 회사 자료 업로드  →  운영 시작

02 — Project

프로젝트 생성

리드와 브로셔를 선택해 캠페인을 만들고 첫 발송을 준비합니다. 캠페인은 초안(draft) → 예약(scheduled) → 발송(sent) → 완료(done) 상태로 흐르며, 생성 즉시 대상 리드 수와 진행 상태가 기록됩니다.

아직 데이터가 없다면 홈은 "오늘의 운영 현황" 빈 상태로 시작합니다. 첫 캠페인 시작하기 버튼으로 캠페인 생성에, 리드 보기로 바이어 목록에 바로 진입할 수 있습니다.

03 — Buyers

바이어 관리

바이어 리드는 관심도(Interest)와 신뢰도(Trust) 두 축으로 평가됩니다. 관심도는 오픈·클릭·체류·답장·미팅 신호를 가중합산(0~100)하고, 신뢰도는 이메일 도메인 품질을 기준으로 감점 방식(0~100)으로 계산합니다.

리드 목록은 즉시 연락 · 검토 필요 · 장기 육성 우선순위로 필터링되며, 관심도가 임계치를 넘으면 핫리드로 자동 표시됩니다. 각 카드는 이름·회사·국가·이메일과 마지막 활동 신호를 함께 보여줍니다.

04 — Messaging

메시지 발송

개인화 변수를 활용해 메시지를 작성하고, 데스크톱·모바일 미리보기로 검증한 뒤 발송합니다. {name}, {company}, {country} 등 7종 변수가 자동 치환되며, 변수를 채울 수 없는 수신자가 있으면 발송 전 경고로 알려 기본값 설정이나 제외를 선택할 수 있습니다.

작성 내용은 3초 디바운스로 자동 저장되고, 1:1 메시지는 1크레딧, 브로셔를 포함하면 5크레딧이 차감됩니다. 발송 후에는 오픈·클릭·답장·미팅 신호가 실시간으로 관심도에 반영됩니다.

05 — Cards

카드형 메시지

명함을 카메라로 촬영하거나 파일로 업로드하면 OCR이 이름·회사·이메일을 자동 수집하고, 상단에서 선택한 박람회 리스트에 곧바로 저장합니다. 가이드 프레임이 명함을 인식하면 4꼭짓점 마커가 활성화되고, 연속 촬영으로 여러 장을 빠르게 등록할 수 있습니다.

수집된 카드는 갤러리에서 박람회 태그로 묶어 관리하며, 그대로 바이어 리드로 전환됩니다. 박람회 카드는 미리보기까지 무료이고, "이걸로 사용" 확정 시점에만 크레딧이 차감됩니다.

명함 등록 흐름+

프레임 정렬 → 촬영(또는 앨범 선택) → OCR 자동 수집 → 박람회 태그 저장 → 리드 전환. 후면 카메라(1920×1080) 기준, 연속 모드 지원.

06 — Brochures

브로셔 활용

회사 브로셔·제안서를 가격정보·제품소개·인증·실적·회사소개 등 분류와 언어별로 업로드해 관리합니다. 업로드된 자료는 자동 분석되어 기본 브로셔 · 추천 자료 · 분석 완료 · 보완 필요 상태로 표시됩니다.

메시지 작성 중에는 자료를 선택해 첨부하거나, AI가 판단한 추천 자료를 바이어에게 제안할 수 있습니다. PDF·DOCX·HTML·PPTX 포맷을 지원하며, 등록 자료·기본 브로셔·추천 자료·보완 필요 수가 요약 지표로 제공됩니다.
